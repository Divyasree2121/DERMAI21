# **App Name**: Derm-AI Navigator

## Core Features:

- User Account Management: Secure sign-up and login functionality to manage user access, including password input validation and toggles.
- Image Upload Interface: A user-friendly interface for uploading dermatoscopic skin images for analysis.
- AI Preprocessing and Analysis Pipeline: Displays real-time feedback on image preprocessing and the ongoing AI model analysis, preparing the image for classification.
- Multi-Class Skin Condition Prediction: Utilizes an EfficientNetB0-based deep learning model to classify uploaded images across 10 different skin conditions.
- AI Prediction Result Display: Presents the AI-predicted skin condition, confidence score, and a mandatory medical disclaimer to the user.
- Problem and Solution Overview: Provides informational content detailing the problem of dermatological diagnosis and how Derm-AI offers a supportive solution, highlighting its technical approach.

## Style Guidelines:

- Primary color (Professional Blue): A deep, trustworthy blue for key interactive elements and branding. Hex: #2E66B3 (HSL: 220, 60%, 45%).
- Background color (Soft Azure Mist): A heavily desaturated, light blue from the primary's hue, providing a clean and expansive canvas. Hex: #EDF3F9 (HSL: 220, 20%, 95%).
- Accent color (Intelligent Violet): An analogous, distinct blue-violet used for secondary actions, highlights, and subtle sophistication. Hex: #553BBE (HSL: 250, 60%, 55%).
- Text and Secondary Elements (Clean Gray): A soft gray for body text, borders, and other subtle UI components to maintain an academic and minimalist feel. Hex: #BFBFBF (HSL: 0, 0%, 75%).
- Headline and body font: 'Inter' (sans-serif) for its modern, neutral, and highly readable qualities, suitable for academic and professional content.
- Minimalist, clean icons consistent with medical and AI themes, such as simplified skin scans, microscopic elements, and subtle neural network patterns.
- A clean, academic layout featuring ample whitespace, clear content hierarchy, and a responsive design that adapts seamlessly to various screen sizes. Page transitions should be smooth and subtle.
- Subtle animations like fade-ins, pulse effects for the logo, and minimal loading spinners or progress bars, enhancing user experience without distraction during transitions and loading states.
